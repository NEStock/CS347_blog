This past week in CS 347 we have learned a lot about Flexbox. According to w3schools, before Flexbox there were four layout modes which included block, inline, table, and positioned. While it may be possible to obtain many layouts using these four layout modes, Flexbox makes creating a variety of layouts much easier.

One common layout that Flexbox enabled is called the Holy Grail Layout. In its simplest form, the Holy Grail layout looks like this:

header

chester

While researching more about Flexbox and the Holy Grail layout, I found that the Holy Grail layout is almost always attributed to Flexbox. However, Flexbox is known for making a wide variety of layout options easily attainable.

Another example of a layout Flexbox enables is actually this paragraph! I first encapsulated this paragraph, the two above, and the one below into one HTML element. Next, I created a parent HTML element to encapsulate the paragraphs into one child and the code section as the other child. Finally, I used Flexbox to place the two children side by side within the parent.

The code I used is shown below.

#holy-grail-code-and-caption {
    display: flex;
    flex-direction: row;
    justify-content: space-between;
    align-items: baseline;
}

#holy-grail-caption {
    flex-basis: 55%;
}

#holy-grail-code-and-caption > pre {
    background-color:rgb(240, 247, 255);
    width: 35%;
    padding: 20px;
    margin-right: 40px;
    font: monospace;
    font-size: 1.1em;
}
#holy-grail-root {
    margin: 40px;
    display: flex;
    flex-direction: column;
    height: 400px;
    border: solid 2px black;
}

#header, #footer {
    background-color: cornflowerblue;
    padding: 30px
}

#middler {
    flex-grow: 1;
    display: flex;
    flex-direction: row;
}

#lefter, #righter {
    background-color: ivory;
    flex-basis: 100px;
}

#chester {
    background-color: lightcoral;
    flex-grow: 1;
    flex-basis: 0;
}

#lefter, #righter, #chester {
    padding: 20px;
}
